
# Astronomy Memory Game

A web application that utilizes randomized images from NASA's Astronomy Photo of the Day (APOD) API as the cards for a memory matching game.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- APOD API
- GitHub Pages
